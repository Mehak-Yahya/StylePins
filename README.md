# StylePins 

StylePins is a Pinterest-inspired fashion discovery and inspiration platform built using React and Firebase. The application allows users to explore visual content, discover fashion ideas, and organize inspiration through a clean and modern interface.

## Features

### Authentication System
- User Registration
- User Login
- Firebase Authentication
- Email Verification
- Protected Routes
- Secure User Sessions

### Pinterest-Style Experience
- Modern Splash Screen
- Pinterest-inspired UI Design
- Responsive Layout
- Visual Content Discovery
- Fashion Inspiration Feed
- Masonry Grid Layout

### User Features
- Personalized Dashboard
- User Profiles
- Account Management
- Verified User Access

## Tech Stack

### Frontend
- React.js
- React Router DOM
- JavaScript (ES6+)
- CSS3

### Backend & Services
- Node.js
- MongodB
- Firebase Authentication
